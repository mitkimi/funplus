# Fun Plus
`funplus` 是为 Focus Latex 网站设计，已简约、酷炫为主要设计的 Vue 2.x 组件库，主要用于满足 Focus Latex 前后台开发使用。

## ✨ Feature
- 📦 开箱即用的高质量、轻量级 Vue 组件
- 🌈 使用较为明确的指向性表达语言
- 🌍 i18n 支持性高
- 🔑 定制化程度高，可复用性强

## 兼容性
- 现代浏览器和 EDGE（抛弃几乎所有版本的 IE，除了 IE11）
- SSR
- [Electron](https://www.electronjs.org/)
